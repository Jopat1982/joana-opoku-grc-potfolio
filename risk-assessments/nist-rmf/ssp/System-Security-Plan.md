# System Security Plan (SSP)

## System Information

**System Name:** ABC Manufacturing Information System

**System Owner:** ABC Manufacturing Corporation

**System Type:** Internal Business Information System

**Security Categorization:** Moderate Impact

---

# System Description

The ABC Manufacturing Information System supports daily business operations, employee management, finance, inventory, and customer services.

---

# System Boundary

The system includes:

- Microsoft 365
- Windows Servers
- Employee Workstations
- Cloud Storage
- Corporate Network
- VPN Remote Access

---

# Security Objectives

### Confidentiality
Protect company and customer information from unauthorized disclosure.

### Integrity
Protect information from unauthorized modification.

### Availability
Ensure business systems remain operational and accessible.

---

# Security Controls

The organization implements security controls based on NIST SP 800-53 including:

- Access Control
- Audit Logging
- Incident Response
- Configuration Management
- Risk Assessment
- Identification and Authentication
- Media Protection
- System Monitoring

---

# Continuous Monitoring

Security controls are reviewed quarterly.

Vulnerability scans are performed monthly.

Annual security assessments are conducted.
